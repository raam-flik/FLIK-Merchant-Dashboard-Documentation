# Checkout Reports

Comprehensive reporting on payment and order performance.

## Overview

Checkout Reports provides detailed insights into your sales and payment processing.

## Checkout Analytics Dashboard

**Screenshot Reference**: `/assets/screenshots/Laporan & Analisis - FLIK Checkout/Laporan & Analisis FLIK Checkout.png`

### Key Metrics

#### Sales Metrics
- **Total Revenue**: Sum of all sales
- **Total Orders**: Order count
- **Average Order Value**: AOV
- **Units Sold**: Total items
- **Conversion Rate**: Visitor to buyer %

#### Payment Metrics
- **Successful Payments**: Completed
- **Failed Payments**: Declined
- **Refunds**: Refunded orders
- **Success Rate**: % successful
- **Pending**: Awaiting payment

#### Invoice Metrics
- **Invoices Created**: Total issued
- **Invoices Paid**: Paid orders
- **Outstanding**: Unpaid invoices
- **Overdue**: Past due invoices
- **Overdue Amount**: $ overdue

#### Form Metrics
- **Forms Created**: Order forms
- **Form Views**: Total views
- **Form Submissions**: Orders via forms
- **Conversion Rate**: View to submit %
- **Abandonment Rate**: Incomplete %

### Report Filters

Filter data by:
- **Date Range**: Custom dates
- **Product**: Specific products
- **Category**: Product category
- **Payment Method**: Card, Bank, etc.
- **Status**: Paid, Pending, Failed

### Visualizations

View data as:
- **Line Chart**: Trends over time
- **Bar Chart**: Comparison data
- **Pie Chart**: Distribution
- **Table**: Detailed list
- **Summary Cards**: Key metrics
