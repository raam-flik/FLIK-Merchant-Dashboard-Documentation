# Financial Module

Manage payments, cash flow, and financial operations for your business.

## Overview

The Financial module provides tools for payment management, account balance tracking, and funding options.

## Features

### 1. Merchant Cash Advance
Access funding for business growth and operations.
- Apply for cash advance
- Track funding status
- Manage repayment
- View funding options
- Application process

[Read Merchant Cash Advance →](./merchant-cash-advance.md)

### 2. Saldo (Balance) Management
Monitor account balance and financial transactions.
- Check available balance
- View transaction history
- Manage payment methods
- Purchase service credits
- Track fund movements

[Read Saldo Documentation →](./saldo.md)

## Quick Start

### Checking Your Balance
1. Go to Saldo
2. View current balance
3. Review recent transactions
4. Check pending payments
5. Monitor account health

### Applying for Funding
1. Go to Merchant Cash Advance
2. Check eligibility
3. Fill application
4. Submit documents
5. Track approval status

## Financial Features

- **Balance Management**: Real-time balance
- **Transaction History**: All movements
- **Credit Purchase**: Buy service credits
- **Payment Methods**: Manage payment sources
- **Fund Requests**: Apply for advance
- **Performance Metrics**: Financial health

## Best Practices

- Monitor balance regularly
- Keep payment methods updated
- Understand fee structure
- Track cash flow
- Plan for growth
