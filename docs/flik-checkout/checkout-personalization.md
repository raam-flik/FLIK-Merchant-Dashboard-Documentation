# Checkout Personalization

Customize your checkout page to match your brand and select available payment and shipping methods.

## Overview

Checkout Personalization allows you to:
- Upload your store logo
- Enable/disable payment methods
- Enable/disable shipping methods
- Configure Facebook Pixel tracking
- Preview checkout page

## Accessing Checkout Personalization

1. Navigate to **FLIK Checkout** from main menu
2. Select **Personalisasi Checkout** (Checkout Personalization)
3. You'll see the personalization form with preview on the right

![Personalization Checkout Page](../../assets/screenshots/FLIK%20Checkout%20-%20Personalisasi%20Checkout%20Page/Personalization%20Checkout%20Page.png)

## Checkout Personalization Sections

### 1. Appearance Section

#### Logo Upload
- **Ubah Logo, format PNG/JPG** (Change logo, PNG/JPG format)
- Click to upload store logo
- Display recommended size: 12x (width x height)
- Current logo displayed above upload link

### 2. Payment Method Section

Select which payment methods customers can use:

#### Available Payment Methods
- **COD** (Cash on Delivery): Pay at delivery
- **Card Payment**: Credit/debit card payments
- **Wallet**: E-wallet options
- **QRIS**: QR code payment
- **Virtual Account**: Bank virtual account transfer
- **Card EDC**: EDC machine card payment

Each payment method has a toggle/checkbox to enable or disable it.

### 3. Shipping Method Section

Select which shipping couriers are available:

#### Available Shipping Methods
- **Anteraja**: Checkbox to enable/disable
- **SiCepat**: Checkbox to enable/disable
- **Grab**: Checkbox to enable/disable
- **J&T**: Checkbox to enable/disable
- **JNE**: Checkbox to enable/disable
- **Gojek**: Checkbox to enable/disable
- **Lion**: Checkbox to enable/disable
- **Lalamove**: Checkbox to enable/disable
- **SAP**: Checkbox to enable/disable
- **Paxel**: Checkbox to enable/disable
- **TIKI**: Checkbox to enable/disable
- **NinjaExpress**: Checkbox to enable/disable
- **ID Express**: Checkbox to enable/disable
- **Custom Ubah**: Custom shipping option

Each courier has a checkbox and dropdown to configure settings.

### 4. CAPI Token Meta Ads Section

Configure Facebook tracking for your checkout:

#### Facebook Pixel Configuration
- **Facebook Pixel ID**: Text field to enter your Facebook Pixel ID
  - Helps track checkout performance on Facebook
  - Link provided: "Click here how to get your Facebook Pixel ID"

#### CAPI Token Configuration
- **CAPI Token**: Text field to enter your Conversion API token
  - Used for server-side tracking
  - Provides insights into your ad campaign performance

## Form Actions

### Bottom Buttons
- **Sebelumnya** (Previous): Go back without saving
- **Selanjutnya** (Next): Continue to next step
- **Submit**: Save all personalization settings

## Preview

The right side of the page shows a live preview of:
- Your uploaded logo
- Checkout header with your branding
- Selected payment methods
- Order summary example
- Total and payment button

## Best Practices

1. ✅ Upload clear, professional logo
2. ✅ Select payment methods relevant to your customers
3. ✅ Enable popular shipping methods in your area
4. ✅ Configure Facebook Pixel for marketing insights
5. ✅ Test checkout before going live
6. ✅ Review preview regularly
7. ✅ Keep shipping methods up to date
8. ✅ Monitor payment method usage

## Tips

- **Logo**: Use PNG/JPG format for best results
- **Payment Methods**: Select methods your customers prefer
- **Shipping Methods**: Include couriers available in your delivery areas
- **Preview**: Check right panel to see how checkout looks
- **Facebook Tracking**: Add Pixel ID to track customer conversions
- **Testing**: Always test checkout with selected methods before publishing

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Logo not uploading | Check file format (PNG/JPG) and size |
| Preview not updating | Refresh page or re-upload logo |
| Payment method not showing | Ensure checkbox is selected |
| Shipping method not available | Enable checkbox for courier |
| Facebook Pixel not tracking | Verify Pixel ID is correct format |
