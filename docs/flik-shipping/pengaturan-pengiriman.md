# Pengaturan Pengiriman (Shipping Settings)

Configure shipping carrier options and services available for your shipments.

## Overview

Pengaturan Pengiriman (Shipping Settings) allows you to:
- Enable/disable shipping carriers
- Select specific shipping services per carrier
- Configure available delivery methods
- Manage carrier preferences

## Accessing Shipping Settings

1. Navigate to **FLIK Shipping** from main menu
2. Select **Pengaturan Pengiriman** or **Shipping Settings**
3. Access the configuration page

## How to Use

1. Review available carriers and their services
2. Check the checkbox next to services you want to enable
3. Uncheck services you want to disable
4. Scroll down to view all available options
5. Click **Simpan** (Save) button at the bottom to save changes

## Save Changes

- **Simpan Button**: Located at bottom right, click to save all configuration changes
