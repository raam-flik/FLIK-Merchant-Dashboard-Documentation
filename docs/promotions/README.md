# Promotions Module

Create and manage promotional campaigns to drive sales and customer engagement.

## Overview

The Promotions module provides tools to create and manage various types of promotional campaigns including vouchers, cashback, QR campaigns, and post-purchase offers.

## Features

### 1. Voucher Management
Create discount codes and voucher campaigns.
- Generate unique voucher codes
- Set discount percentage or fixed amount
- Define expiration dates
- Track redemption
- Analyze performance

[Read Voucher Documentation →](./voucher.md)

### 2. Cashback Programs
Offer cashback incentives to customers.
- Create cashback rules
- Set reward percentages
- Target specific products/categories
- Track payouts
- Manage customer rewards

[Read Cashback Documentation →](./cashback.md)

### 3. QR Campaign
Create QR code-based marketing campaigns.
- Generate QR codes
- Link to promotions
- Track scan metrics
- Share via digital/print
- Measure engagement

[Read QR Campaign Documentation →](./qr-campaign.md)

### 4. Post Purchase Offer
Encourage repeat purchases with post-purchase offers.
- Create time-limited offers
- Target recent buyers
- Set discount terms
- Track conversion
- Maximize customer lifetime value

[Read Post Purchase Offer Documentation →](./post-purchase-offer.md)

## Quick Start

### Creating Your First Promotion

1. **Choose Promotion Type**
   - Voucher for broad audience
   - Cashback for customer loyalty
   - QR Campaign for marketing
   - Post-Purchase for repeat sales

2. **Set Up Details**
   - Campaign name and description
   - Discount/reward amount
   - Expiration date
   - Target audience (optional)

3. **Configure Rules**
   - Minimum purchase amount
   - Product restrictions
   - Usage limits
   - Single/multiple use

4. **Launch Campaign**
   - Activate promotion
   - Share campaign code/link
   - Monitor performance
   - Make adjustments

## Best Practices

- Start small and test
- Monitor performance daily
- Adjust based on results
- Combine multiple promotions
- Clear communication
- Track ROI carefully
- Update inventory regularly
- Customer transparency
