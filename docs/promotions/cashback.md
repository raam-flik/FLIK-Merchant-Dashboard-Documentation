# Cashback Programs

Create cashback incentive programs to reward customer loyalty and encourage repeat purchases.

## Overview

Cashback Programs allow you to offer monetary rewards to customers for purchases.

### Creating Cashback Rule

**Screenshot Reference**: `/assets/screenshots/Promosi - Cashback/Create New Cashback Rule.png`

#### Program Details
- **Program Name**: Name of cashback program
- **Description**: What customers get
- **Start Date**: Program launch date
- **End Date**: Program expiration
- **Status**: Active or Inactive

#### Reward Settings
- **Cashback Type**: Percentage or Fixed Amount
- **Reward Value**: Amount or percentage
- **Maximum Reward**: Per transaction limit
- **Minimum Purchase**: Qualifying amount

#### Application Rules
- **Product Categories**: Which products
- **Specific Products**: Individual items
- **All Products**: Everything included
- **Exclusions**: What's not eligible

#### Redemption
- **Wallet Credit**: Add to customer account
- **Direct Refund**: Return to payment method
- **Future Purchase Credit**: Use on next order
- **Manual Redemption**: Customer requests

### Manage Cashback Programs
- Monitor total rewards issued
- Track participation
- View redemption rates
- Adjust reward amounts
- Pause/Resume programs

## Customer Communication

- Announce program launch
- Explain how it works
- Show reward calculations
- Track earned cashback
- Enable redemption

## Analytics

Track program effectiveness:
- Total rewards issued
- Redemption rate
- ROI calculation
- Repeat purchase rate
- Customer satisfaction
