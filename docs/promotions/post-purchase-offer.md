# Post Purchase Offer

Create and manage post-purchase offers to encourage repeat purchases.

## Overview

Post Purchase Offer allows you to create promotional offers targeted at customers based on their previous purchases.

## Post Purchase Offer List Page

![List of Post Purchase Offer](../../assets/screenshots/Promosi%20-%20Post%20Purchase%20Offer/List%20of%20Post%20Purchase%20Offer.png)

### Summary Statistics

The page displays three key metrics at the top:

- **Total Pendapatan** (Total Revenue): Total revenue from post-purchase offers (e.g., Rp0)
- **Conversion Rate**: Overall conversion rate percentage (e.g., 1%)
- **Total Penawaran Diterima** (Total Offers Received): Total number of offers accepted (e.g., 1)

### Search Section

#### Search Bar
- **Search field**: "Cari nama produk" (Search by product name)
- Search icon to find specific products

#### Action Button
- **Buat Post Purchase Offer** (Create Post Purchase Offer): Button to create new offer

### Post Purchase Offer Table

#### Table Columns
- **Nama Produk** (Product Name): Name of the product being offered (with sort icon)
  - Example: "Aerostreet T-Shirt Oversize Pop Team Hitam Kaos Fadaa"
- **Product Upsell**: The upsell product being offered (with sort icon)
  - Example: "Aerostreet Massive Low Costa Gum Hitam Orange"
- **Dilihat** (Viewed): Number of times offer was viewed (with sort icon)
  - Example: 0
- **Diterima** (Accepted): Number of times offer was accepted (with sort icon)
  - Example: 0
- **Ditolak** (Rejected): Number of times offer was rejected (with sort icon)
  - Example: 0

#### Additional Columns (Missing in current doc)
- **Konversi** (Conversion): Conversion rate or number for the offer
- **Pendapatan** (Revenue): Revenue generated from the offer
- **Status** (Status): Current status of the offer (Active/Inactive)
- **Action**: Edit button to modify the PPO

## Create Post Purchase Offer Page

![Create Post Purchase Offer](../../assets/screenshots/Promosi%20-%20Post%20Purchase%20Offer/Create%20Post%20Purchase%20Offer.png)

### Form Section: Cari & Pilih Produk (Search & Select Product)

#### Product Selection
- **Cari nama produk** (Search product name): Dropdown field with placeholder "Ketik untuk mencari produk"
- **Status Tawaran Produk** (Product Offer Status): Toggle switch with two options:
  - "Tidak Aktif" (Inactive)
  - "Aktif" (Active)

#### Product List Section
- **Daftar Produk yang Ditawarkan** (List of Offered Products): Area showing products to be offered
- **Tambah Tawaran Produk** (Add Product Offer): Button to add more products to the offer

### Action Buttons
- **Kembali** (Back): Return to previous page
- **Buat Post Purchase Offer** (Create Post Purchase Offer): Create the offer

## How to Use

### Create New Post Purchase Offer
1. Click **Buat Post Purchase Offer** button on List page
2. Search for product name in "Cari nama produk" field
3. Select product from dropdown
4. Toggle "Status Tawaran Produk" to set offer status (Aktif/Tidak Aktif)
5. Add additional products if needed using "Tambah Tawaran Produk" button
6. Click **Buat Post Purchase Offer** to create

### Edit Post Purchase Offer
1. Click edit action button next to offer in list
2. Modify offer details
3. Save changes

### View Offer Performance
1. Check "Dilihat" (Viewed) count - how many customers saw the offer
2. Check "Diterima" (Accepted) count - how many accepted the offer
3. Check "Ditolak" (Rejected) count - how many rejected the offer
4. View "Konversi" (Conversion) rate
5. Check "Pendapatan" (Revenue) generated
6. Monitor "Status" to ensure offer is active
