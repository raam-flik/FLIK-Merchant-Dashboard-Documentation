# Voucher Management

Create and manage discount voucher campaigns to drive sales and customer engagement.

## Overview

Voucher Management allows you to create promotional codes and discounts for customers.

![List Voucher Page](../../assets/screenshots/Promosi%20-%20Voucher/List%20Voucher%20Page.png)

View all created vouchers with status, usage, and performance metrics.

### Creating a Voucher

![Buat Voucher Page](../../assets/screenshots/Promosi%20-%20Voucher/Buat%20Voucher%20Page.png)

#### Basic Information
- **Voucher Code**: Unique discount code
- **Voucher Name**: Display name
- **Description**: What the voucher offers
- **Start Date**: When voucher becomes active
- **End Date**: When voucher expires

#### Discount Settings
- **Discount Type**: Percentage or Fixed Amount
- **Discount Value**: Amount or percentage
- **Maximum Discount**: Max discount limit
- **Minimum Purchase**: Minimum order value

#### Usage Rules
- **Usage Limit**: Total vouchers available
- **Per Customer**: Limit per customer
- **One-time Use**: Can only use once
- **Stackable**: Can combine with other vouchers

#### Targeted Conditions (Optional)
- **Applicable Products**: Specific or all
- **Product Categories**: Which categories
- **Customer Segment**: Who can use
- **Geographic**: Specific areas

### Manage Vouchers
- View usage statistics
- Edit active vouchers
- Pause/Resume campaigns
- Delete expired vouchers
- Export voucher data

## Best Practices

- Clear, memorable codes
- Realistic expiration dates
- Monitor redemption rates
- Set appropriate minimums
- Test before full launch
